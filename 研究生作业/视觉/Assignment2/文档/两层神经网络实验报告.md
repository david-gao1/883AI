# 两层全连接神经网络实验报告

## 摘要

本实验实现了一个两层全连接神经网络（Two-Layer Fully-Connected Neural Network），并在CIFAR-10数据集上进行了图像分类任务。网络架构包括：输入层 → 全连接层1 → ReLU激活 → 全连接层2 → Softmax损失。实验重点完成了前向传播、反向传播、损失计算和梯度计算的实现，使用随机梯度下降（SGD）优化，并通过网格搜索在验证集上确定了最优的超参数（隐藏层大小=300，学习率=1e-3，正则化强度=0.05）。实验结果表明，默认参数下（hidden_size=50）验证集准确率为28.7%，说明模型容量不足；通过超参数调优（36种组合），最优组合达到验证集55%，测试集54.7%的分类准确率，相比线性分类器（SVM 37-39%，Softmax 33.8%）有显著提升（+15-20个百分点）。实验验证了非线性模型（ReLU激活）的表达能力，以及超参数调优在模型性能中的关键作用。

**关键词**：神经网络、全连接层、ReLU激活、反向传播、随机梯度下降、超参数调优、CIFAR-10

## 1 实验目的

本实验旨在实现一个完整的两层全连接神经网络，重点完成以下目标：

1. **理解神经网络的基本原理**：前向传播、反向传播、梯度计算
2. **掌握ReLU激活函数**：理解非线性激活函数在神经网络中的作用
3. **实现反向传播算法**：理解链式法则在梯度计算中的应用
4. **学习超参数调优**：在验证集上搜索最优的隐藏层大小、学习率和正则化强度
5. **熟悉神经网络训练流程**：在CIFAR-10数据集上完成端到端的分类任务

## 2 神经网络架构

### 2.1 网络结构

**架构**：
```
输入 (N, D) → 全连接层1 (D → H) → ReLU → 全连接层2 (H → C) → Softmax损失
```

其中：
- $N$：batch大小
- $D = 3072$：输入维度（32×32×3图像展平）
- $H$：隐藏层大小（可调超参数）
- $C = 10$：类别数

**参数**：
- $W_1$：第一层权重矩阵，形状$(D, H)$
- $b_1$：第一层偏置向量，形状$(H,)$
- $W_2$：第二层权重矩阵，形状$(H, C)$
- $b_2$：第二层偏置向量，形状$(C,)$

### 2.2 前向传播

**第一层**：
$$z_1 = X \cdot W_1 + b_1$$
$$a_1 = \text{ReLU}(z_1) = \max(0, z_1)$$

**第二层**：
$$scores = a_1 \cdot W_2 + b_2$$

**实现代码**（`cs231n/classifiers/neural_net.py`）：
```python
# First layer: fully connected + ReLU
z1 = X.dot(W1) + b1  # (N, H)
a1 = np.maximum(0, z1)  # ReLU: (N, H)

# Second layer: fully connected
scores = a1.dot(W2) + b2  # (N, C)
```

### 2.3 损失函数

**Softmax损失（交叉熵损失）**：
$$L_i = -\log\left(\frac{e^{s_{y_i}}}{\sum_j e^{s_j}}\right)$$

**正则化项（L2）**：
$$R(W) = \lambda \left(\sum_{i,j} W_{1,ij}^2 + \sum_{i,j} W_{2,ij}^2\right)$$

**总损失**：
$$L = \frac{1}{N}\sum_i L_i + R(W)$$

**实现代码**：
```python
# Softmax loss with numerical stability
exp_scores = np.exp(scores - np.max(scores, axis=1, keepdims=True))
probs = exp_scores / np.sum(exp_scores, axis=1, keepdims=True)

# Cross-entropy loss
correct_logprobs = -np.log(probs[np.arange(N), y])
data_loss = np.sum(correct_logprobs) / N

# Regularization loss
reg_loss = reg * (np.sum(W1 * W1) + np.sum(W2 * W2))

loss = data_loss + reg_loss
```

### 2.4 反向传播

**梯度计算**（链式法则）：

**Softmax损失梯度**：
$$\frac{\partial L}{\partial scores} = \frac{\partial}{\partial scores} \left(-\log(p_{y_i})\right) = p - \mathbf{1}_{y_i}$$

其中$p$是softmax概率，$\mathbf{1}_{y_i}$是one-hot编码的正确类别。

**第二层梯度**：
$$\frac{\partial L}{\partial W_2} = a_1^T \cdot \frac{\partial L}{\partial scores} + 2\lambda W_2$$
$$\frac{\partial L}{\partial b_2} = \sum_{i=1}^N \frac{\partial L}{\partial scores_i}$$

**ReLU梯度**：
$$\frac{\partial a_1}{\partial z_1} = \begin{cases} 1 & \text{if } z_1 > 0 \\ 0 & \text{if } z_1 \leq 0 \end{cases}$$

**第一层梯度**：
$$\frac{\partial L}{\partial a_1} = \frac{\partial L}{\partial scores} \cdot W_2^T$$
$$\frac{\partial L}{\partial z_1} = \frac{\partial L}{\partial a_1} \odot \mathbf{1}_{z_1 > 0}$$
$$\frac{\partial L}{\partial W_1} = X^T \cdot \frac{\partial L}{\partial z_1} + 2\lambda W_1$$
$$\frac{\partial L}{\partial b_1} = \sum_{i=1}^N \frac{\partial L}{\partial z_{1,i}}$$

**实现代码**：
```python
# Gradient from softmax loss
dscores = probs.copy()
dscores[np.arange(N), y] -= 1
dscores /= N

# Backprop through second layer
grads['W2'] = a1.T.dot(dscores) + 2 * reg * W2
grads['b2'] = np.sum(dscores, axis=0)

# Backprop through ReLU
da1 = dscores.dot(W2.T)
dz1 = da1.copy()
dz1[z1 <= 0] = 0  # ReLU gradient

# Backprop through first layer
grads['W1'] = X.T.dot(dz1) + 2 * reg * W1
grads['b1'] = np.sum(dz1, axis=0)
```

## 3 训练过程

### 3.1 随机梯度下降（SGD）

使用SGD优化损失函数，每次迭代随机采样batch计算梯度：

```python
def train(self, X, y, X_val, y_val,
          learning_rate=1e-3, learning_rate_decay=0.95,
          reg=5e-6, num_iters=100,
          batch_size=200, verbose=False):
    for it in range(num_iters):
        # 随机采样batch
        batch_indices = np.random.choice(num_train, batch_size, replace=True)
        X_batch = X[batch_indices]
        y_batch = y[batch_indices]
        
        # 计算损失和梯度
        loss, grads = self.loss(X_batch, y=y_batch, reg=reg)
        
        # 更新权重
        self.params['W1'] -= learning_rate * grads['W1']
        self.params['b1'] -= learning_rate * grads['b1']
        self.params['W2'] -= learning_rate * grads['W2']
        self.params['b2'] -= learning_rate * grads['b2']
        
        # 每个epoch衰减学习率
        if it % iterations_per_epoch == 0:
            learning_rate *= learning_rate_decay
```

**SGD的优势**：
- 比全批量梯度下降快，适合大规模数据
- 随机性有助于跳出局部最优
- 内存占用小，只需存储batch数据

**学习率衰减**：
- 每个epoch后，学习率乘以衰减率（0.95）
- 帮助模型在后期更精细地优化

### 3.2 梯度检查

使用数值梯度检查验证解析梯度的正确性：


## 4 超参数调优

### 4.1 默认参数训练（Baseline）

**默认参数设置**：
- 隐藏层大小：`50`
- 学习率：`1e-4`

- 正则化强度：`0.25`
- 迭代次数：`1000`

**训练结果**：

**问题分析**：
1. **模型容量不足**：隐藏层大小50太小，无法学习复杂的特征组合
2. **训练不充分**：迭代次数1000可能不够，损失仍在下降
3. **学习率可能偏小**：损失下降缓慢，可能需要更大的学习率或更多迭代

**改进方向**：
- 增加隐藏层大小（100-500）
- 增加迭代次数（3000）
- 调整学习率和正则化强度

### 4.2 调优策略

采用网格搜索（Grid Search）在验证集上搜索最优超参数：

**搜索空间**：
- **隐藏层大小（hidden_size）**：[100, 200, 300, 500]
- **学习率（learning_rate）**：[1e-3, 5e-4, 1e-4]
- **正则化强度（regularization strength）**：[0.05, 0.1, 0.25]

## 6 关键发现与讨论

### 6.1 ReLU激活函数的重要性

**ReLU的优势**：
- **非线性**：使模型能够学习非线性决策边界
- **计算简单**：$\max(0, x)$计算快速
- **梯度稳定**：在正区域梯度为1，避免梯度消失

**对比线性分类器**：
- 线性分类器（SVM、Softmax）只能学习线性决策边界
- 神经网络通过ReLU激活能够学习非线性决策边界
- 这是神经网络性能提升的关键因素

### 6.2 超参数的影响

**隐藏层大小**：
- **太小**（如100）：模型容量不足，欠拟合
- **太大**（如500）：容易过拟合，需要更强的正则化
- **合适**（如300）：在表达能力和泛化能力之间取得平衡

**学习率**：
- **太大**：训练不稳定，可能发散
- **太小**：训练慢，可能陷入局部最优
- **配合衰减**：初始较大学习率快速收敛，后期较小学习率精细优化

**正则化强度**：
- **太小**：过拟合，训练集准确率高但验证集低
- **太大**：欠拟合，训练集和验证集准确率都低
- **合适**：防止过拟合同时保持模型容量

### 6.3 Inline Question分析

**Inline Question**：如何减少训练集和测试集准确率差距？

**答案**：选项1（在更大的数据集上训练）和选项3（增加正则化强度）

**解释**：
- **问题本质**：训练集准确率高但测试集低是**过拟合**的表现
- **解决方案1**：更大的数据集提供更多样化的样本，帮助模型学习更通用的特征
- **解决方案3**：更强的正则化限制权重大小，防止模型过度拟合训练数据
- **错误选项2**：增加隐藏单元会增大模型容量，加剧过拟合
