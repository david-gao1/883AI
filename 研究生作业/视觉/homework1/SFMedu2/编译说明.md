# VLFeat ARM64 编译说明

## 方法1：使用MATLAB脚本编译（推荐）

1. **在MATLAB中运行**（确保MATLAB是ARM64版本）：
   ```matlab
   cd('/Users/lianggao/MyWorkSpace/002install/matlab/视觉/Assignment1/SFMedu2')
   compile_all_arm64
   ```

2. **等待编译完成**（可能需要几分钟）

3. **验证编译结果**：
   ```matlab
   mexext  % 应该显示 mexmaca64
   vl_setup('quiet')
   exist('vl_sift', 'file')  % 应该返回 3（MEX文件）
   ```

4. **运行主程序**：
   ```matlab
   SFMedu2
   ```

## 方法2：使用Makefile编译

1. **打开终端**，运行：
   ```bash
   cd /Users/lianggao/MyWorkSpace/002install/matlab/视觉/Assignment1/SFMedu2
   ./compile_vlfeat_arm64.sh
   ```

2. **等待编译完成**

3. **在MATLAB中验证**：
   ```matlab
   cd('/Users/lianggao/MyWorkSpace/002install/matlab/视觉/Assignment1/SFMedu2')
   vl_setup('quiet')
   SFMedu2
   ```

## 注意事项

- 确保MATLAB是ARM64版本（运行 `mexext` 应该显示 `mexmaca64`）
- 编译需要MATLAB的MEX编译器正常工作
- 如果编译失败，检查MATLAB的编译器设置：`mex -setup`
- 编译过程可能需要几分钟时间

## 故障排除

如果编译失败：
1. 检查MATLAB版本和架构：`version` 和 `mexext`
2. 检查MEX编译器：`mex -setup`
3. 查看错误信息，可能需要安装Xcode命令行工具：
   ```bash
   xcode-select --install
   ```
