
================================================================================
多视角人脸图片获取指南 - 问题2
================================================================================

由于直接下载可能受限，请按照以下步骤获取真实的多视角人脸图片：

方法1: 使用 AFLW2000 数据集（推荐）
--------------------------------------------------------------------------------
1. 访问网站: http://www.cbsr.ia.ac.cn/users/xiangyuzhu/projects/3DDFA/main.htm
2. 下载 AFLW2000 数据集
3. 从数据集中选择同一人的多张不同角度照片（至少5张，建议8-12张）
4. 将选中的图片复制到: images/face_reconstruction/
5. 重命名为: face_01.jpg, face_02.jpg, ..., face_12.jpg
6. 运行 MATLAB 脚本: SFMedu2_face.m

方法2: 使用 300W-LP 数据集
--------------------------------------------------------------------------------
1. 访问 GitHub: https://github.com/YadiraF/PRNet
2. 下载或克隆仓库
3. 在 TestImages 文件夹中找到测试图片
4. 选择同一人的多张不同角度照片
5. 复制到: images/face_reconstruction/
6. 重命名并运行脚本

方法3: 使用 LFW 数据集
--------------------------------------------------------------------------------
1. 访问: http://vis-www.cs.umass.edu/lfw/
2. 下载 LFW 数据集
3. 选择同一人的多张照片（LFW中同一人通常有多张照片）
4. 复制到: images/face_reconstruction/
5. 重命名并运行脚本

方法4: 使用自动化脚本
--------------------------------------------------------------------------------
1. 下载任意公开数据集
2. 运行: python images/face_reconstruction/prepare_images.py
3. 按照提示选择图片
4. 脚本会自动复制和重命名

图片要求：
- 至少5张图片（建议8-12张）
- 同一人的不同角度照片
- 照片之间有足够的重叠区域
- 建议包含：正面、左侧、右侧、上方等不同角度
- 格式：JPG 或 PNG

快速开始：
1. 下载数据集
2. 选择图片
3. 运行: python images/face_reconstruction/prepare_images.py
4. 运行: SFMedu2_face.m
