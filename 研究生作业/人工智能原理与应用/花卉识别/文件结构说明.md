
## 🚀 使用方法

### 运行训练

从项目根目录运行：

```bash
cd code
python main.py
```

或者从项目根目录：

```bash
python code/main.py
```

### 注意事项

1. **路径说明**：代码文件在 `code/` 文件夹中，数据文件夹 `data/` 在项目根目录
2. **配置文件**：`config.py` 中的路径已经更新，使用相对路径 `../data` 访问数据
3. **输出文件**：训练结果（模型、图片、JSON）会保存在项目根目录

## 📝 文件说明

### code/ 文件夹
包含所有Python源代码文件，采用模块化设计，便于维护和扩展。

### docs/ 文件夹
包含所有文档和说明文件，包括：
- 项目说明文档
- 代码解释文档
- 实验报告
- 优化方案和效果分析

### data/ 文件夹
包含训练、验证和测试数据集。
