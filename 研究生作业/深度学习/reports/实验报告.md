# CIFAR-100 分类实验报告

## 1. 实验目标

本实验基于 CIFAR-100 数据集完成图像分类任务，分别使用 MLP（多层感知机）和 CNN（卷积神经网络）两类基础网络，系统比较不同学习率（0.1、0.01、0.001）与不同优化器（SGD、Adam）对模型训练效果的影响，共计 12 组对比实验，并对测试结果进行可视化分析。

## 2. 实验环境

| --- | --- |
| 操作系统 | macOS (Apple Silicon) |

| 框架 | PyTorch ≥ 2.2、TorchVision ≥ 0.17 |
| 其他依赖 | NumPy、Matplotlib、Pandas、tqdm |
| 包管理 | uv |

## 3. 数据集与预处理

CIFAR-100 数据集包含 100 个类别，每张图片大小为 32×32 像素，共 50,000 张训练图像和 10,000 张测试图像。

本实验的数据划分方式：
- 从训练集中划出 10% 作为验证集（5,000 张），剩余 45,000 张用于训练
- 测试集保持原始 10,000 张不变

预处理策略：
- **训练集**：随机裁剪（padding=4）、随机水平翻转、标准化（均值 [0.5071, 0.4867, 0.4408]，标准差 [0.2675, 0.2565, 0.2761]）
- **验证集与测试集**：仅标准化

## 4. 模型设计

### 4.1 MLP（多层感知机）

将 32×32×3 的图像拉平为 3,072 维向量后输入全连接网络，结构如下：

| 层 | 输出维度 | 激活/操作 |
| --- | --- | --- |
| Flatten | 3072 | — |
| Linear | 1024 | ReLU + Dropout(0.3) |
| Linear | 512 | ReLU + Dropout(0.3) |
| Linear | 100 | — |

MLP 不利用图像的空间结构信息，作为基础对比模型。

### 4.2 CNN（卷积神经网络）

使用三组卷积块逐步提取多尺度空间特征，结构如下：

| 阶段 | 层 | 通道/维度 | 特征图尺寸 |
| --- | --- | --- | --- |
| Block 1 | Conv3×3 → BN → ReLU → Conv3×3 → ReLU → MaxPool2×2 → Dropout2d(0.3) | 3→64→64 | 32→16 |
| Block 2 | Conv3×3 → BN → ReLU → Conv3×3 → ReLU → MaxPool2×2 → Dropout2d(0.3) | 64→128→128 | 16→8 |
| Block 3 | Conv3×3 → BN → ReLU → MaxPool2×2 | 128→256 | 8→4 |
| Classifier | Flatten → Linear(4096, 512) → ReLU → Dropout(0.3) → Linear(512, 100) | 256×4×4→100 | — |

CNN 通过局部卷积核捕获空间特征，BatchNorm 稳定训练，Dropout 防止过拟合。

## 8. 结果分析

### 8.1 模型结构的影响

CNN 在所有对应配置下均优于 MLP，最大优势达 18 个百分点。原因在于：
- CNN 的卷积核共享权重，参数量远少于全连接层，降低了过拟合风险
- 卷积操作保留了空间局部性，能够捕获边缘、纹理等低级视觉特征
- 多层池化实现了平移不变性，提高了泛化能力
- MLP 将图像拉平后丢失了所有空间结构信息

### 8.2 学习率的影响

| 学习率 | 表现 |
| --- | --- |
| **0.01** | SGD 的最优学习率，训练稳定且收敛较快 |
| **0.001** | Adam 的最优学习率；对 SGD 略偏小，收敛稍慢但仍有效 |
| **0.1** | 对 Adam 完全不可用（发散至随机水平）；SGD 配合 CNN 勉强可用但效果欠佳，配合 MLP 则严重震荡 |

学习率 0.1 导致 Adam 优化器下的所有实验测试准确率均降至 1%，等同于在 100 类上的随机猜测，这是因为 Adam 的自适应学习率机制会放大过大的初始步长，导致参数更新剧烈振荡。

### 8.3 优化器的影响

- **SGD（+Momentum）**：在较宽的学习率范围内（0.001~0.01）表现稳健，是图像分类中的经典选择。动量项帮助跨越局部极小值，梯度方向更稳定。
- **Adam**：仅在 lr=0.001 时有效，lr≥0.01 时迅速发散。Adam 的自适应特性意味着它对初始学习率更加敏感——过大的学习率会被自适应机制进一步放大。

### 8.4 训练轮数的影响

从最佳实验（CNN+SGD+lr=0.01）的训练曲线来看，验证准确率在第 10 个 epoch 仍在持续上升（从 12.58% 到 36.32%），loss 持续下降，没有出现过拟合或收敛停滞。这表明 10 个 epoch 的训练尚未充分，增加训练轮数（如 50~100 epoch）并配合学习率衰减策略，预计能显著提升最终精度。

## 9. 实验总结

本实验完成了基于 CIFAR-100 的 12 组分类对比实验，主要结论如下：

1. **CNN 显著优于 MLP**：在最优超参数配置下，CNN 测试准确率（38.69%）比 MLP（20.45%）高出近一倍，验证了卷积结构对图像空间特征提取的关键作用。
2. **学习率是最敏感的超参数**：不当的学习率可直接导致训练失败（准确率降至 1%），合理的学习率选择比优化器选择更重要。
3. **SGD 泛化更稳定**：SGD+Momentum 在较宽的学习率范围内均可有效训练，而 Adam 仅在小学习率下可用。
4. **10 个 epoch 不足以充分训练**：最佳模型在第 10 轮仍处于上升阶段，未达收敛。

后续改进方向：
- 增加训练轮数（50~100 epoch），配合余弦退火或阶梯式学习率衰减
- 引入更深的网络架构（如 ResNet）或预训练模型
- 增强数据增强策略（如 Cutout、MixUp）
- 使用学习率预热（Warmup）策略缓解初始阶段不稳定问题
